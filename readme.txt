== Description ==
StreamTest.Net Badges is a simple plugin for displaying StreamTest badges anywhere on your WordPress site.


== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `streamtestbadges.zip` in the 'Add New Plugin' admin menu
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings > Streamtest Badges in the WordPress Admin menu
4. Copy and paste any of the shortcodes to anyhwere in your WordPress website.
